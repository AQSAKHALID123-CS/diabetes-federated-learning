# Diabetes Prediction using Federated Learning

## Description
This project implements a federated learning-based approach for diabetes prediction using medical datasets.

## Features
- Federated Learning model
- Centralized baseline comparison
- Performance evaluation (Accuracy, ROC, etc.)

## How to Run
1. Open the notebook in Google Colab
2. Install dependencies
3. Run all cells

- Additional dataset (if used)

## Results
